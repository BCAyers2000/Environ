API
===

Environ exposes an API that can be used by host programs to integrate
Environ into their calculations. This can be achieved by importing the
``environ`` instance of the API from ``environ_api`` as follows::

   USE environ_api, ONLY: environ

The following links describe the API's exposed objects and methods
accessible via the ``environ`` instance:
